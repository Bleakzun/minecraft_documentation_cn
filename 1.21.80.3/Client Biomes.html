概述

Minecraft client_biome 文件定义资源包中生物群系的客户端设置。
这是过去在 biomes_client.json 中按生物群系设置的新首选位置。
（从基础游戏版本 1.21.40 开始，biomes_client.json不再从内置的原版资源包加载。无论版本如何，该文件仍会为其他内容加载，并且使用旧版基础游戏的世界也仍会使用它。）
这些文件不是“自定义生物群系”实验玩法的一部分，不会导致生物群系定义独立存在，但它们可用于自定义生物群系的客户端设置。

纲要

客户端生物群系组件

此客户端生物群系使用的任何组件

名称 类型 必填？ 描述
minecraft:ambient_sounds 对象 自选 设置生物群系的环境声音。这些声音必须位于“sounds.json”文件的“individual_named_sounds”中。
minecraft:atmosphere_identifier 对象 自选 [内部开发中] 设置延迟渲染期间使用的大气环境参数。未配置此组件的生物群系将采用默认大气设置。
minecraft:biome_music 对象 自选 音乐在生物群系中的播放方式
minecraft:color_grading_identifier 对象 自选 [内部开发中] 配置延迟渲染时使用的色彩分级(color_grading)参数。未包含此组件的生物群系将使用默认色彩分级设置。
minecraft:dry_foliage_color 对象 自选 设置渲染时使用的干枯植被颜色。未配置此组件的生物群系将采用默认的干枯植被颜色表现。
minecraft:fog_appearance 对象 自选 设置渲染期间使用的迷雾设置。没有此组件的生物群系将使用默认的雾设置。
minecraft:foliage_appearance 对象 自选 设置渲染期间使用的树叶颜色或颜色贴图。没有此组件的生物群系将具有默认的树叶外观。
minecraft:grass_appearance 对象 自选 设置渲染期间使用的草地颜色或颜色贴图。没有此组件的生物群系将具有默认的草外观。
minecraft:lighting_identifier 对象 自选 [内部开发中] 设置延迟渲染时使用的光照参数。未配置此组件的生物群系将采用默认光照设置。
minecraft:sky_color 对象 自选 设置渲染期间使用的天空颜色。没有此组件的生物群系将使用默认的天空颜色。
minecraft:water_appearance 对象 自选 设置渲染期间使用的水面颜色。没有此组件的生物群系将使用默认的水面颜色。
minecraft:water_identifier 对象 自选 [内部开发中] 配置延迟渲染使用的水体参数。未启用此组件的生物群系将使用默认水体设置。

客户端生物群系定义

包含用于定义客户端生物群系的描述和组件。

名称 类型 必填？ 描述
components 客户端生物群系组件类型的对象 必填 此客户端生物群系的组件。
description 客户端生物群系组件类型的对象 必填 非组件设置，包括客户端生物群系名称。

客户端生物群系描述

包含客户端生物群系的非组件设置。

名称 类型 必填？ 描述
identifier 字符串 必填 客户端生物群系的名称，由其他功能（如 '/locate biome' 命令）使用。必须与游戏或行为包定义的生物群系的名称匹配。

minecraft:ambient_sounds

设置生物群系的环境声音。这些声音必须位于“sounds.json”文件的“individual_named_sounds”中。

名称 类型 必填？ 描述
addition 字符串 自选 偶尔在听者位置播放的声音
loop 字符串 自选 当听者位置位于生物群系内时循环的声音
mood 字符串 自选 当光照水平较低时，很少在附近的空气方块位置播放的声音。没有环境声音的生物群系将使用 'ambient.cave' 声音。
返回顶部

minecraft:atmosphere_identifier

[内部开发中] 设置延迟渲染期间使用的大气环境参数。未配置此组件的生物群系将采用默认大气设置。

名称 类型 必填？ 描述
atmosphere_identifier 字符串 必填 要使用的大气环境定义标识符。
返回顶部

minecraft:biome_music

音乐在生物群系中的播放方式

名称 类型 必填？ 描述
music_definition 字符串 自选 在该生物群系内时播放的音乐。如果未指定或未找到，则默认音乐将由维度决定。空字符串将导致没有音乐。
volume_multiplier 浮点数 自选 在此生物群系中时，会暂时并逐渐应用于音乐音量的乘数。必须是介于 0 和 1 之间的值（包括 0 和 1）。
