Recipes

配方在 behavior_packs /'name of pack'/ recipes 目录下的 Json 文件中设置。
配方 JSON 文件具有不同的结构，具体取决于其类型。

熔炉配方

表示熔炉的熔炉配方。'input' 物品将燃烧并转化为 'output' 中指定的物品。

参数

名称 类型 描述
input 物品名称 用作熔炉配方输入的物品。
output 物品名称 用作熔炉配方输出的物品。

有序配方

表示工作台上的有序合成配方。
模式中使用的键可以是除 ' 空格 ' 字符以外的任何单个字符，该字符是为配方中的空槽保留的。

参数

名称 类型 描述
assume_symmetry 布尔值 确定配方是否应采用对称性。
key 键和物品对的数组 字符映射到物品名称。
pattern 字符串的数组 字符，这些字符表示要由键定义的模式。
priority 整数 用作配方输出的物品
result 物品名的数组 配方的输出结果。
tags 字符串的数组 表示配方在哪里被使用

锻造转化（锻造升级）配方

表示锻造台的锻造转化配方
此配方会将一个物品转换为另一个物品，同时保留其属性。

参数

名称 类型 描述
addition 物品名称 执行升级操作所需的材质。在可堆叠物品的情况下，只消耗 1 件物品。唯一接受的物品是 "minecraft:netherite_ingot"。物品必须带有 "minecraft:transform_materials" 标签才能被放到相应的 UI 槽中。
base 物品名称 要转换的物品。其属性将被复制到 “result” 中。唯一接受的物品是盔甲和工具。物品必须带有 "minecraft:transformable_items" 标签才能被放到相应的 UI 槽中。
result 物品名称 要转换为的物品。
tags 字符串的数组 与配方兼容的工作方块。唯一接受的值是 “smithing_table”。
template 物品名称 执行转换操作所需的模板。在可堆叠物品的情况下，只消耗 1 件物品。物品必须带有 “minecraft:transform_templates” 标签才能被放到相应的 UI 槽中。

锻造装饰配方

表示锻造台的锻造装饰配方
此配方将彩色装饰图案应用于物品，同时保留其其他属性。

参数

名称 类型 描述
addition 物品名称或物品标签 执行装饰操作所需的材料。它定义将图案应用于物品的颜色。在可堆叠物品的情况下，只消耗 1 件物品。物品必须带有 “minecraft:trim_materials” 标签才能被接受到相应的 UI 槽中。
base 物品名称或物品标签 要装饰的物品。其属性将被保留。唯一接受的物品是盔甲。物品必须带有 “minecraft:trimmable_armors” 标签才能被接受到相应的 UI 槽中。
tags 字符串的数组 与配方兼容的工作方块。唯一接受的值是 “smithing_table”。
template 物品名称或物品标签 执行装饰操作所需的模板。它定义将应用于物品的样式。在可堆叠物品的情况下，只消耗 1 件物品。物品必须带有 “minecraft:trim_templates” 标签才能被接受到相应的 UI 槽中。
